# 🍅 Tomato Leaf Disease Classification

## 📌 Project Overview

This project focuses on detecting and classifying the health condition of tomato plants based on leaf images. The model is trained to recognize **10 different classes**, including both healthy leaves and various diseases.

The goal of this project is to support early detection of plant diseases, which can help improve crop quality and reduce agricultural losses.

---

## 🧠 Model Details

* Type: Image Classification
* Number of classes: 10
* Input: Leaf images of tomato plants
* Output: Predicted class label (disease or healthy)

---

## 📊 Classes

The model can detect the following conditions:

* Healthy
* Bacterial Spot
* Early Blight
* Late Blight
* Leaf Mold
* Septoria Leaf Spot
* Spider Mites
* Target Spot
* Mosaic Virus
* Yellow Leaf Curl Virus
